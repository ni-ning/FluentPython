
本项目为《流程的Python》学习代码笔记，使用工具为 [Jupyter Notebook](https://jupyter.readthedocs.io/en/latest/install.html)

#### 安装步骤
```bash
pip install jupyter
jupyter notebook    # 自动跳转 http://localhost:8888/tree
```

#### 常用快捷键
```
a: 向下插入一个cell
b: 向上插入一个cell
m: 将cell的类型切换为markdown类型
y: 将cell的类型切换为code类型
shift+enter: 执行cell
shift+tab: help文档
```
